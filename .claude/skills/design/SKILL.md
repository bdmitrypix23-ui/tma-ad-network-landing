# Дизайн гайд: креативность + антислоп

## Главный принцип

Будь **смелым в концепции**, сдержанным в декоре. Не бойся необычного layout, яркого CSS-арта, характерных шрифтов — бойся шаблонных gradient overlay, одинаковых карточек и pill buttons. Креативность и антислоп — не противоположности. Креативность = делай уникальное. Антислоп = не делай шаблонное.

## Перед тем как писать код

При создании UI с нуля — задай себе 3 вопроса:
1. **Настроение** — какую эмоцию вызывает интерфейс? (уютный, дерзкий, элегантный, технический, игривый)
2. **Одна запоминающаяся деталь** — что пользователь запомнит? CSS-арт? Необычный layout? Яркий шрифт? Если ответ "ничего" — придумай.
3. **Палитра шрифтов** — display + body, оба характерные, НЕ Inter/Roboto/Arial.

## Рабочий процесс

### Как субагент (тебя вызвали через Agent tool)

Читай **только `references/good-patterns.md`**. Blacklist НЕ для тебя — антислоп-правила уже в чеклисте промпта. Твоя задача — креативность: уникальные иконки, характерные шрифты, атмосферные фоны, оркестрированные анимации. Не бойся деталей — filled иконки лучше outline, сложный CSS-арт лучше минимализма.

### Как основной агент (пишешь CSS сам)

1. **Прочитай `references/good-patterns.md`** — сначала секцию Креативность, потом Рецепты.

2. **Найди реальный HTML** — открой TSX/JSX компонент, посмотри какие теги и классы рендерятся. НЕ УГАДЫВАЙ селекторы.

3. **Проверь по чёрному списку** — прочитай `references/blacklist.md`. Сверь каждое правило. 23 забаненных паттерна с готовыми заменами.

4. **Один акцентный приём на блок** — не gradient + shadow + border + glow одновременно.

## Быстрые правила (не читая references)

- **Hover** — внутри `@media (hover: hover)`. Subtle, не dramatic.
- **Transition** — на конкретное свойство, никогда `all`.
- **Border-radius** — пропорционален размеру: button 4-6px, card 8px, modal 12px.
- **Тени** — только для elevation (dropdown, modal). Карточки = border.
- **Emoji** — не использовать как иконки. CSS-арт или SVG вместо.
- **Glass** — максимум 1-2 элемента на странице.
- **Шрифты** — всегда fallback. Всегда уникальная пара, не дефолты.
